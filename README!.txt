CARA PASANG

1.  Pastikan sudah menginstall:

-   PHP
-   Composer
-   Node.js
-   (Serta dependency pendukung lainnya)

2.  Jalankan proses instalasi Klik file “INSTAL.BAT”.

Saat instalasi berjalan, akan muncul perintah: “Masukan lokasi folder”

Isi dengan lokasi folder project web Anda. Contoh: C: Lalu tekan Enter.

3.  File otomatis dibuat Setelah instalasi selesai, akan muncul dua
    file:

-   START.BAT
-   CHECK_VERSION.BAT

4.  Sebelum menjalankan START Klik “CHECK_VERSION.BAT” untuk memastikan
    PATH sudah terpasang dengan benar.

5.  Setup database Pastikan Anda sudah mengatur konfigurasi database di
    file .env.
